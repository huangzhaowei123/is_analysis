# 接口：getTest  [返回](../README.md)
用例： [发布实验](../yongli/发布实验.md)

- 功能：
    发布一个新的实验。
    
- 权限：
    老师：可以发布新的实验。
    
- API请求地址： 
    接口基本地址/v1/api/getTest/<teacher_id>

- 请求方式 ：
    GET

- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |teacher_id|老师的编号|
    
- 返回实例：

        {         
           "title":"实验三";
           "content":"实验内容：1.**************
                               2.**************
                               3.**************";
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |title|实验标题|
  |content|实验内容|
